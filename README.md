# gudang_app

A Flutter project for warehouse.

## Getting Started


Aplikasi ini untuk mengontrol stok gudang dapur MBG basis Android dan web.

Fitur

Multi user
Anda bisa membuat user admin, staff, penerima barang, pengolah bahan dan supalayer dengan hak akses berbeda.

Daftar stok gudang,
menampilkan stok yang berada di gudang.

Memasukan stok manual,
Bisa memasukan barang langsung tanpa pesanan.

Mengeluarkan stok manual,
Mengeluarkan stok untuk diproses atau kadaluarsa.

Memasukan stok otomatis dari menu,
Barang/bahan otomatis masuk pesanan jika di gudang kosong, setelah anda membuat menu dg komposisi yang sudah di racik (bumbu).

Memperlihatkan pesanan yang berjalan,
Daftar pesanan masuk untuk hari ini atau berdasarkan tanggal bisa di pantau dimanapun, berapa yang masuk, siapa yang menerima.

Export Import data stok, Pesanan barang,
Import Data Excel barang yang sudah ada di gudang, Export untuk di cetak.
